== Description ==
Universal content parser for autofilling blogs, news websites, online catalogues, etc.
The parser covers all types of content: text, images and videos.

Plugin works through the external service wpparser.com, which guarantees the best performance.
We do not collect nor transfer any personal data, except anonymized statistics regarding the usage of the service. For more information see terms of use http://wpparser.com/terms-and-conditions/


### Features ###
* Get articles from any website and post it on your resource;
* Set up many links simultaneously;
* Grab records from several pages of the website;
* Filters helps to exclude unwanted data;
* Simple settings – just mark the title and the page’s content.



== Installation ==

1. Place plugin files to the `/wp-content/plugins/`folder.
2. Activate the plugin on the Plugin page in Administrator Panel.
